# 🧩 组件库

现代化、易用且高性能的 React 组件库，基于 **shadcn/ui** 构建，助力开发者快速构建美观、响应式的 Web 应用。

## ✨ 特性

- 🚀 **丰富组件** - 40+ 基础组件 + 高级扩展组件
- 🎨 **完全可定制** - 基于 Tailwind CSS，支持主题定制
- 📦 **零依赖** - 按需加载，体积轻量
- ♿ **无障碍支持** - 符合 WCAG 2.1 标准
- 🌗 **深色模式** - 原生支持明暗主题切换
- 🛠️ **TypeScript** - 完整的类型定义
- 📱 **响应式设计** - 完美适配各种设备
- ⚡ **高性能** - 优化的渲染性能

## 🚀 快速开始

### 安装依赖

```bash
npm install @radix-ui/react-* lucide-react class-variance-authority clsx tailwind-merge
```

### 基础使用

```tsx
import { Button } from "@/components/ui/button"
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card"

export default function App() {
  return (
    <Card className="w-[350px]">
      <CardHeader>
        <CardTitle>欢迎使用组件库</CardTitle>
      </CardHeader>
      <CardContent>
        <Button>开始使用</Button>
      </CardContent>
    </Card>
  )
}
```

## 🎨 设计系统

### 色彩系统

我们的设计系统基于语义化的色彩命名，支持明暗主题：

```css
/* 主色调 */
--primary: 222.2 84% 4.9%;
--primary-foreground: 210 40% 98%;

/* 背景色 */
--background: 0 0% 100%;
--foreground: 222.2 84% 4.9%;

/* 卡片 */
--card: 0 0% 100%;
--card-foreground: 222.2 84% 4.9%;

/* 弹出层 */
--popover: 0 0% 100%;
--popover-foreground: 222.2 84% 4.9%;

/* 边框 */
--border: 214.3 31.8% 91.4%;

/* 输入框 */
--input: 214.3 31.8% 91.4%;

/* 主要操作 */
--primary: 222.2 47.4% 11.2%;
--primary-foreground: 210 40% 98%;

/* 次要操作 */
--secondary: 210 40% 96%;
--secondary-foreground: 222.2 84% 4.9%;

/* 强调色 */
--accent: 210 40% 96%;
--accent-foreground: 222.2 84% 4.9%;

/* 破坏性操作 */
--destructive: 0 84.2% 60.2%;
--destructive-foreground: 210 40% 98%;

/* 环形进度条 */
--ring: 222.2 84% 4.9%;

/* 半径 */
--radius: 0.5rem;
```

## 📖 使用指南

### 1. 组件导入

```tsx
// 基础组件
import { Button } from "@/components/ui/button"
import { Input } from "@/components/ui/input"

// 扩展组件
import { Breadcrumb } from "@/components/extensions/breadcrumb/breadcrumb"
import { Pagination } from "@/components/extensions/pagination/pagination"
```

### 2. 主题配置

```tsx
// 在 globals.css 中配置主题
@layer base {
  :root {
    --background: 0 0% 100%;
    --foreground: 222.2 84% 4.9%;
    /* ... 其他变量 */
  }

  .dark {
    --background: 222.2 84% 4.9%;
    --foreground: 210 40% 98%;
    /* ... 深色主题变量 */
  }
}
```

### 3. 响应式设计

```tsx
// 使用 Tailwind 的响应式前缀
<Card className="w-full md:w-1/2 lg:w-1/3">
  <CardContent className="p-4 md:p-6">
    {/* 内容 */}
  </CardContent>
</Card>
```

### 4. 无障碍访问

```tsx
// 组件已内置无障碍支持
<Button aria-label="提交表单">
  提交
</Button>

<Input aria-describedby="email-help" />
<div id="email-help">请输入有效的邮箱地址</div>
```

## 🎯 最佳实践

### 性能优化

- 使用 `React.memo` 优化组件重渲染
- 合理使用 `useMemo` 和 `useCallback`
- 按需导入组件，避免全量引入

### 可访问性

- 为交互元素提供 `aria-label`
- 使用语义化的 HTML 标签
- 确保键盘导航支持
- 提供足够的颜色对比度

### 样式定制

- 优先使用 CSS 变量进行主题定制
- 利用 Tailwind 的工具类快速开发
- 保持设计系统的一致性
