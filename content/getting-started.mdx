# 🚀 快速开始

本指南将帮助您快速上手组件库，在几分钟内开始构建您的应用。

## 📋 前置要求

在开始之前，请确保您的开发环境满足以下要求：

- **Node.js** 18.0 或更高版本
- **npm** 或 **pnpm** 或 **yarn** 包管理器
- 基本的 **React** 和 **TypeScript** 知识

## 🛠️ 安装

### 1. 创建项目

如果您还没有 React 项目，可以使用以下命令创建一个：

```bash
# 使用 Next.js (推荐)
npx create-next-app@latest my-app --typescript --tailwind --eslint

# 或使用 Vite
npm create vite@latest my-app -- --template react-ts
```

### 2. 安装依赖

在项目根目录下运行以下命令安装必要的依赖：

```bash
npm install @radix-ui/react-accordion @radix-ui/react-alert-dialog @radix-ui/react-aspect-ratio @radix-ui/react-avatar @radix-ui/react-checkbox @radix-ui/react-collapsible @radix-ui/react-context-menu @radix-ui/react-dialog @radix-ui/react-dropdown-menu @radix-ui/react-hover-card @radix-ui/react-label @radix-ui/react-menubar @radix-ui/react-navigation-menu @radix-ui/react-popover @radix-ui/react-progress @radix-ui/react-radio-group @radix-ui/react-scroll-area @radix-ui/react-select @radix-ui/react-separator @radix-ui/react-slider @radix-ui/react-slot @radix-ui/react-switch @radix-ui/react-tabs @radix-ui/react-toggle @radix-ui/react-toggle-group @radix-ui/react-tooltip lucide-react class-variance-authority clsx tailwind-merge
```

### 5. 创建工具函数

创建 `lib/utils.ts` 文件：

```ts
import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

## 🔧 添加更多组件

### 基础组件

基础组件位于 `components/ui/` 目录下，您可以直接导入使用：

```tsx
import { Button } from "@/components/ui/button"
import { Input } from "@/components/ui/input"
import { Label } from "@/components/ui/label"
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card"
```

### 扩展组件

扩展组件位于 `components/extensions/` 目录下，提供更高级的功能：

```tsx
import { Breadcrumb, BreadcrumbItem } from "@/components/extensions/breadcrumb/breadcrumb"
import { Pagination } from "@/components/extensions/pagination/pagination"
```

## 🎨 主题定制

### 自定义颜色

您可以通过修改 CSS 变量来自定义主题颜色：

```css
:root {
  --primary: 220 14% 96%; /* 自定义主色调 */
  --primary-foreground: 220 9% 46%;
}
```

### 自定义组件样式

使用 `cn` 工具函数组合样式：

```tsx
import { cn } from "@/lib/utils"
import { Button } from "@/components/ui/button"

<Button className={cn(
  "bg-blue-500 hover:bg-blue-600",
  "text-white font-bold"
)}>
  自定义按钮
</Button>
```

## 📱 响应式设计

组件库完全支持响应式设计，使用 Tailwind CSS 的响应式前缀：

```tsx
<Card className="w-full md:w-1/2 lg:w-1/3">
  <CardContent className="p-4 md:p-6">
    <h3 className="text-lg md:text-xl font-semibold">响应式标题</h3>
    <p className="text-sm md:text-base">响应式内容</p>
  </CardContent>
</Card>
```

## ♿ 无障碍访问

所有组件都内置了无障碍访问支持：

```tsx
<Button aria-label="提交表单">
  提交
</Button>

<Input 
  aria-describedby="email-help"
  aria-required="true"
/>
<div id="email-help">请输入有效的邮箱地址</div>
```
